üìã Setup Instructions

  1. Create a Notion Integration: Go to notion.so/my-integrations and create a new integration. Copy the token.
  2. Create a Database: In Notion, create a database with these properties:
    • Name (Title) - for content names
    • Canva URL (URL) - link to your Canva design
    • Image (Files & Media) - optional fallback for images/videos
    • Order (Number) - for grid position
    • Type (Select) - Options: "Photo", "Reel", "Carousel"
    • Slides (Files & Media) - for carousel images (multiple files, optional)
  3. Share Database: Click "..." ‚Üí "Add connections" ‚Üí Select your integration
  4. Get Database ID: Copy the ID from the database URL
  5. Add Content:
    • Using Canva: Paste any Canva design URL in the "Canva URL" field. The widget will automatically detect and display photos, videos, or carousels!
    • Using Direct Files: Add images/videos to "Image" or "Slides" fields as before
    • Priority: Canva URL is used first if available, then falls back to uploaded files
  6. Canva URL Format: Use embed URLs like:
    https://www.canva.com/design/[ID]/view?embed

üé® Canva Tips

  • To get the embed URL: Open your Canva design ‚Üí Click "Share" ‚Üí "More" ‚Üí "Embed" ‚Üí Copy the URL from the code
  • Make sure your Canva design is set to "Anyone with the link"
  • Canva videos and carousels will display automatically based on your design
  • The widget shows a Canva badge for designs loaded from Canva

Note: This widget stores your API credentials in memory only. They're not saved anywhere.
